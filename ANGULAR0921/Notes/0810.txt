Angular is one of the preferred framework for building web applications.

Latest version of Angular is 12.

In order to create Angular Project we have multiple ways and the most recommended way is to use Angular CLI.

Command Line Interface

We need to install CLI using

npm install -g @angular/cli
	|
we can use angular commands to create project and also project content throughtout the development time.
